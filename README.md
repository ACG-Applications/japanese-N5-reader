# 🎌 JLPT N5 Reading & Comprehension App

## ✨ Features

### 📖 Reading
- **10 Story Blocks** - Follow Kenji's journey through Tokyo
- **100 Unique Kanji** - Learn essential JLPT N5 Kanji in context
- **Furigana Toggle** - Show/hide reading guides with ruby `<rt>` tags
- **Translation Toggle** - Show/hide English translations
- **Grammar Highlighting** - Color-coded grammar patterns in text

### 🎧 Listening
- **Text-to-Speech (TTS)** - Native Japanese voice support
- **Speed Control** - Adjust speed: 0.75x, 1.0x, 1.5x
- **Word-by-Word Highlighting** - Yellow highlight follows the audio
- **Progress Bar** - Visual track of TTS progress

### ❓ Comprehension
- **Multiple Choice Quizzes** - 3 questions per block
- **Smart Distractors** - Plausible wrong answers that make sense
- **Immediate Feedback** - Correct/incorrect with explanations
- **Progress Tracking** - Score and accuracy tracking

### 🎮 Grammar Detective Game
- **Pattern Identification** - Identify grammar patterns in sentences
- **Difficulty Levels** - Easy → Medium → Hard (auto-adjusts)
- **Combo System** - Streak bonuses for correct answers
- **Hint System** - Limited hints available per question
- **Score Tracking** - Points, combos, and achievements

### 📊 Analytics
- **Pattern Mastery** - Track progress for each grammar pattern
- **Strength Scores** - 0-100 for each pattern
- **Weak Pattern Detection** - Identify what needs review
- **Study Recommendations** - Personalized suggestions
- **Study Streak** - Track daily consistency
- **Achievement System** - Earn badges for milestones

### 🎨 Interface
- **Dark Mode** - Study comfortably at night
- **Responsive Design** - Works on desktop, tablet, and mobile
- **Font Size Control** - Small, medium, large
- **Keyboard Shortcuts** - Navigate without mouse
- **Print Support** - Print-friendly styles
